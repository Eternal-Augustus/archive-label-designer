# 档案标签排版器

  <img src="./华北电力大学-logo.svg" alt="华北电力大学校徽" width="96" />
</p>

面向档案整理场景的离线桌面应用。它用于把学生信息 Excel 批量排版成 A4 标签页，在应用内完成模板设计、分页预览和打印确认，再交给 Windows 系统打印或 `Microsoft Print to PDF` 输出。

## 功能概览

- Excel 导入：读取首个工作表，自动识别表头和数据行。
- 动态字段映射：不同年级、不同培养层次、不同表头结构都可以重新绑定。
- 行单元模板：每个字段按“整行单元格”方式铺满单格，更接近档案袋标签常见的表格观感。
- 默认模板：内置 `4 x 3` A4 布局，以及“编号 / 姓名 / 班级 / 学号 / 状态”这一套默认模板。
- 编号合成：支持“入学年份 + 博/硕 + 流水号”的编号规则。
- 模板保存 / 加载：当前版式可保存为 JSON，后续直接复用。
- 应用内打印预览：先核对分页、边线、字号和换行，再发送到系统。
- 完全离线：Excel 数据、模板文件和打印流程都在本机完成。

## 典型使用流程

1. 打开应用，导入学生信息 Excel。
2. 检查左侧字段库，确认表头是否读取正确。
3. 在模板区调整单格内部的行顺序、字体、字号、字重、对齐和溢出处理。
4. 根据标签纸尺寸微调页面布局，例如列数、行数、页边距和边线。
5. 打开“打印与输出”，在应用内先核对分页效果。
6. 选择实体打印机直接输出，或选择 `Microsoft Print to PDF` 保存为 PDF。

## 当前默认模板

默认模板已内置以下内容：

- 页面布局：A4，`4 x 3`
- 外层标签块：白底黑字，标签块边线清晰，适合后续人工裁剪
- 行内容：
  - `编号`：按 `25-博-001` 这类格式拼接
  - `姓名`
  - `班级`
  - `学号`
  - `状态`

默认字段绑定基于当前示例表头：

- `field_1` -> 流水号
- `field_4` -> 姓名
- `field_5` -> 学号
- `field_7` -> 班级
- `field_27` -> 状态

如果导入的 Excel 列结构不同，应用会优先尝试自动匹配，也可以在右侧手动重新绑定。

## 输出说明

当前项目采用“两段式”输出策略：

- 预览：在应用内部完成，保证分页和样式检查更直观。
- 实际输出：交由 Windows 系统打印完成。

这意味着：

- 连接实体打印机时，可以直接发送打印任务。
- 需要 PDF 文件时，推荐选择 `Microsoft Print to PDF`。
- 项目当前不再保留单独的“直接导出 PDF”按钮，以避免预览与实际输出之间出现渲染差异。

## 运行方式

### Windows 使用

当前可直接使用的目录版压缩包位于：

```text
dist/archive-label-designer-win-unpacked.zip
```

使用方式：

1. 在 Windows 上解压压缩包。
2. 进入解压后的 `win-unpacked` 目录。
3. 运行 `Archive Label Designer.exe`。

注意：

- 不能只单独拷走 `Archive Label Designer.exe` 运行，它依赖同目录下的 `resources`、`locales` 和 DLL 文件。
- 如果选择 `Microsoft Print to PDF`，保存文件仍由 Windows 系统窗口继续接管。

### 本地开发

```bash
npm install
npm run dev
```

说明：

- `npm run dev` 会同时启动 Vite 与 Electron。
- 浏览器单独打开渲染页时，会自动载入一份演示数据，便于先调整模板和分页观感。
- 真正的 Excel 文件读取、模板文件存取和系统打印，需要在桌面客户端环境中验证。

### 构建

```bash
npm run build
```

### Windows 目录版打包

```bash
npm run dist:win:dir
```

### Windows 安装包打包

```bash
npm run dist:win
```

说明：

- 在缺少 `wine` 的 Linux 环境中，NSIS 安装包可能无法完整产出。
- 目录版通常更适合当前项目的分发和测试。

## 当前定位

这不是一个在线系统，而是一套适合档案室、办公室内网或离线电脑使用的桌面工具。它的重点不是复杂权限管理，而是：

- 快速导入 Excel
- 快速调整标签版式
- 保证打印前可见
- 尽量减少手工转置、拼接和重复排版
